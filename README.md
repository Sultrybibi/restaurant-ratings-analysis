# Restaurant Ratings Analysis

## Project Overview
This project analyzes restaurant rating data to identify key factors that influence customer satisfaction.

## Objectives
- Analyze restaurant ratings across different cities
- Examine the impact of price range on customer ratings
- Explore how smoking policy affects overall satisfaction

## Tools Used
- Microsoft Excel (data cleaning, analysis, pivot tables, charts)
- Google Docs (final reporting)

## Key Insights
- Mid-priced restaurants generally received higher average ratings
- Non-smoking restaurants showed better customer satisfaction
- Customer ratings varied significantly across cities
